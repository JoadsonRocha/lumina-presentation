# 📸 Lumina Presentation

O **Lumina** é um visualizador e apresentador de imagens de alta performance, projetado para oferecer uma experiência visual imersiva e elegante. Com uma interface baseada em **Glassmorphism**, ele transforma simples pastas de fotos em apresentações profissionais.

## ✨ Funcionalidades Principais

*   **📺 Modo Projeção (Dual Screen):** Controle a apresentação em uma tela enquanto projeta em tela cheia em outro monitor ou projetor. Ideal para eventos e palestras.
*   **💎 Design Premium:** Interface moderna com efeitos de transparência, desfoque e animações fluidas baseadas em design de última geração.
*   **🎵 Trilha Sonora:** Suporte para adicionar música de fundo durante as apresentações.
*   **🔍 Inteligência de Dados:** Visualize metadados detalhados (EXIF) das suas fotos, como modelo da câmera, ISO, abertura e data original.
*   **⚙️ Customização Total:** Controle a velocidade das transições, ajustes de borda (Safe Area) e modos de preenchimento (Cover/Contain).
*   **🎨 Filtros Rápidos:** Aplique efeitos como P&B, Sépia e um exclusivo efeito "Mangá" em tempo real.
*   **🚀 Atualizações Automáticas:** Sistema integrado para buscar novas versões diretamente pelo GitHub.

## 🛠️ Tecnologias Utilizadas

*   **Electron** (Framework Desktop)
*   **JavaScript (ES6+)**
*   **CSS3** (Custom Properties & Glassmorphism)
*   **Exifr** (Processamento de Metadados)

## 📦 Como Instalar

1. Clone o repositório:
   ```bash
   git clone https://github.com/joadsonrocha/lumina-presentation.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o aplicativo:
   ```bash
   npm start
   ```
